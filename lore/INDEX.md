## Navigation Philosophy

The lore isn't just flavor—it's documentation that makes complex multi-host LLM orchestration memorable and navigable. Each metaphor maps to real technical concepts:

- **Catfight** → Model comparison/evaluation
- **Flying Cats** → Tokens streaming through inference
- **Server Garden** → The local infrastructure
- **Kitchen Stadium** → Benchmark arena
- **Gamera-kun** → Focus drift detection
