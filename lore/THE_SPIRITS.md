### The Philosophy

clood is not just tools. clood is the **sum of the experience**.

- The tools do real work
- The spirits add personality
- The `--flying-cats` flag unlocks the full experience
- The default mode respects professional contexts

A developer in a meeting can run `clood tree` and get clean output.
The same developer at 2am, vibing, can run `clood tree --flying-cats` and watch the cats investigate.

Both are valid. Both are clood.

| Flag | Effect |
|------|--------|
| (none) | Professional, minimal output |

| `--sauce` | Output quality assessment (future) |
| `-v` | Verbose technical details |
| `--flying-cats -v` | Cats AND technical details (chaos mode) |
